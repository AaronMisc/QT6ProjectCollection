# QT6 Project Collection
A collection of projects made in Qt 6. Made for learning C++.

## Usage
**Actions**: Click "Actions" on the menu bar to open a drop down menu with a list of actions. Then you can press on the action to run it or use the keyboard shortcut.
**Pages**: Click on a menu button in the menu bar, then in the drop down, click a page to go to it.

## Projects
**Main**: Information about this entire project.
**MiniProjects**: Small single-page projects. Information about these projects are on their page.
